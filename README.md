# Heart Insurance Prediction

A machine learning project analyzing heart disease and insurance data to build predictive models.

## Contents

- **Heart.ipynb** - Heart disease prediction analysis and model
- **insurance.ipynb** - Insurance data analysis and modeling
- **heart.csv** - Heart disease dataset
- **insurance.csv** - Insurance dataset
